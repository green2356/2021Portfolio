넥슨코리아

Pleatform : PC, Mobile
Language : HTML5, CSS3, Jquery

Problems

기존 웹 사이트는 쇼핑몰 사이트의 특성상 이미지가 많이 첨부되어있습니다.
이미지를 크게 볼 수 있도록 이미지 배열을 3열로 배치하였습니다.
그러므로 홈페이지 로딩시간이 길어지고, 페이지의 길이가 길어 가독성이
떨어진다는 문제점이 있습니다.
사이트 사용자들에 맞는 쉽고 빠르게 정보를 접할 수가 없는 웹사이트 입니다.

Solutions

웹표준과 접근성을 준수하며, 기존의 자료들은 그대로 유지하면서 리뉴얼합니다.
페이지 길이를 줄일 수 있도록, 배열 3열에서 4열로 바꿉니다.
복잡하지 않으면서도 가독성을 향상하기 위해 쉬운 아이콘과 글씨체, 색을 사용합니다.
